** You are using the OneCore Linux OSKit addon. (version 25H1)
**   First you want to rebase the OS using the latest supported TinyCore ISO :
**    'osrebase latest' - Rebasing from source is not yet supported !
**   To build all SKUs as ISOs you should do :
**    'build -media:cdrom -skus:all'
**   To create a new SKU (rel_main2) :
**    'osmanage createsku rel_main2 "Release 2"'
